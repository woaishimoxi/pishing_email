# 邮件详情页功能说明

## 新增功能概述

当用户需要查看当前检测邮件的各个部分具体内容时，可以点击右上角的**"原始邮件内容"**按钮，展开详细的邮件各部分内容展示面板。

## 功能特性

### 1. 邮件头信息 (Headers)
- 以表格形式展示所有邮件头字段
- 包括：From, To, Subject, Date, Message-ID, Authentication-Results 等
- 支持点击"复制"按钮快速复制所有内容

### 2. 纯文本正文 (Text Body)
- 显示邮件的纯文本版本内容
- 自动处理换行和格式
- 支持一键复制到剪贴板

### 3. HTML 正文 (HTML Body)
- 显示邮件的 HTML 版本内容（转义后展示）
- 防止 XSS 攻击的安全处理
- 支持复制 HTML 代码

### 4. URL 列表
- 列出从邮件中提取的所有 URL
- 标记 HTTP/HTTPS 协议类型（红色为 HTTP 高风险，绿色为 HTTPS）
- 显示 URL 序号

### 5. 附件列表
- 显示所有附件的详细信息
- 包括：文件名、类型、大小、MD5 哈希值
- 标注可疑附件（如双重扩展名、可执行文件等）

### 6. 原始完整邮件
- 展示完整的原始邮件内容（包括所有头部和正文）
- 使用只读文本框展示
- 支持一键复制全部内容

## UI 交互设计

### 手风琴折叠面板
采用 Bootstrap Accordion 组件，每个部分都可以独立展开/收起：
- **默认展开**: 邮件头信息
- **默认收起**: 其他所有部分
- **一键收起**: 右上角提供"收起所有"按钮

### 快捷操作
每个板块底部提供复制按钮：
```
[i 复制] 复制
```
点击即可将内容复制到系统剪贴板。

## 使用方法

1. **触发条件**: 需要先进行一次邮件检测（粘贴内容或上传文件）
2. **操作方式**:
   - 在检测结果页面的右上角，点击"原始邮件内容"按钮
   - 或者滚动到页面下方找到"邮件各部分详情"区域
3. **查看详情**: 点击各个板块的标题展开详细内容
4. **关闭页面**: 再次点击"原始邮件内容"按钮或"收起所有"按钮

## 技术实现

### 前端
- 使用 Bootstrap 5 Accordion 组件
- JavaScript 函数 `toggleRawEmail()` 控制显示/隐藏
- `renderRawEmailContent()` 负责填充数据
- `copyToClipboard()` 实现剪贴板功能
- `escapeHtml()` 防止 XSS 攻击

### 后端
- API `/api/upload` 和 `/api/analyze` 返回 `raw_email` 字段
- 包含完整的解析数据：`parsed.urls`, `parsed.body`, `parsed.html_body`, `headers`, `attachments`

## 安全考虑

1. **XSS 防护**: 所有 HTML 内容都会经过转义处理
2. **只读展示**: 原始邮件内容为只读状态，不可编辑
3. **无网络请求**: 详情页完全本地渲染，不发送额外网络请求

## 适用场景

- 安全分析师需要深入分析邮件结构
- 用户想了解邮件被检测出的具体原因
- 需要进行邮件取证和证据保留
- 学习邮件协议和 Headers 结构
