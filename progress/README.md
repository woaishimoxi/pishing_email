## 🚀 系统概述

**面向中小型企业的轻量化钓鱼邮件检测与溯源系统**

一个集高精度检测、自动化溯源、可视化管理和一键式部署于一体的综合性钓鱼邮件安全防护解决方案。

### 核心特性
- **轻量高效**：2 核 CPU、4GB 内存即可流畅运行
- **高精度检测**：基于机器学习模型，支持演示模式
- **多维度分析**：融合邮件头、URL、文本内容及威胁情报
- **自动溯源**：追踪邮件传输路径和 URL 跳转链路
- **可视化界面**：直观的 Web 管理后台
- **一键部署**：Docker Compose 快速上线

## 🔧 技术架构

### 后端框架
- **Python 3.9+**
- **Flask** - Web 服务框架
- **LightGBM** - 机器学习分类器
- **SQLite** - 数据存储

### 前端框架
- **Bootstrap 5.3** - UI 框架
- **Chart.js** - 数据可视化
- **Bootstrap Icons** - 图标库

### 依赖库
- `email-parser`: 邮件解析
- `whois`: 域名信息查询
- `requests`: HTTP 请求
- `numpy`: 数值计算
- `scikit-learn`: 机器学习工具

## 📦 快速开始

### Docker 部署 (推荐)
```bash
cd code
./start.sh        # Linux/Mac
# 或双击 start.bat  # Windows
```

访问地址：http://localhost:5000

### 本地开发
```bash
pip install -r requirements.txt
python -m src.app
```

## 🎯 API 接口
