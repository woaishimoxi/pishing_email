# 系统更新记录 - v1.2.0

## 更新日期
2026-03-08

## 更新内容

### 1. API 配置管理功能
- 添加了可视化的 API 配置管理界面
- 支持配置 VirusTotal API 的 API Key 和 API URL
- 支持配置 IP-API 的 API URL
- 提供 API 连接测试功能
- 配置自动持久化到 `config/api_config.json` 文件

### 2. 代码优化
- 重构了所有 API 调用相关的函数，添加了可配置的 API URL 参数
- 优化了配置加载和保存机制
- 改进了错误处理和用户反馈

### 3. 文档更新
- 更新了 README.md 中的配置说明部分
- 添加了 API 配置管理的详细说明
- 更新了项目结构和版本信息

## 技术细节

### 后端修改
- `app.py`：添加了 API 配置管理路由和配置文件处理
- `features.py`：添加了 `vt_api_url` 参数支持
- `email_traceback.py`：添加了 `ip_api_url` 参数支持

### 前端修改
- `dashboard.html`：添加了 API 配置模态框和相关功能

## 使用说明

1. 启动系统后，点击导航栏中的 "API 配置" 按钮
2. 在弹出的模态框中输入相应的 API 配置
3. 点击 "测试连接" 按钮验证配置是否正确
4. 点击 "保存配置" 按钮保存更改
