# 系统更新记录 - v1.3.0

## 更新时间
2026-03-08

## 更新内容

### 1. 附件分析功能升级 - 动静结合的多层防御体系

#### 核心改进
- **新增沙箱分析模块** (`sandbox_analyzer.py`)：集成 VirusTotal API，实现动态文件分析
- **增强附件特征提取**：从 5 维扩展到 8 维，新增沙箱分析相关特征
- **多层防御策略**：实现 "静态初筛 -> 动态深析 -> 综合判定" 的三层机制

#### 技术实现细节

##### 新增模块：`sandbox_analyzer.py`
- **功能**：负责与在线沙箱服务交互，执行文件动态分析
- **支持服务**：VirusTotal API（首选）
- **分析流程**：
  1. 先查询文件哈希（秒级响应）
  2. 哈希未命中时上传文件进行分析（异步）
  3. 轮询获取分析结果

##### 优化附件解析：`parse_email.py`
- **保留文件内容**：在解析附件时保留 `content` 字段，用于沙箱分析
- **增强附件信息**：完整提取文件名、类型、大小、哈希值等信息

##### 增强特征工程：`features.py`
- **新增沙箱特征**：
  - `sandbox_detected`：沙箱是否检测到恶意行为
  - `max_sandbox_detection_ratio`：最大检测率
  - `has_sandbox_analysis`：是否进行了沙箱分析
- **智能分析触发**：仅对高风险文件类型进行沙箱分析

##### 集成到 Web 服务：`app.py`
- **传递 API 密钥**：将 VirusTotal API 密钥传递给特征提取函数
- **优化响应**：移除附件内容字段以减少响应大小
- **添加分析状态**：在返回结果中包含沙箱分析状态

#### 分析策略

| 阶段 | 触发条件 | 执行动作 | 优势 |
| :--- | :--- | :--- | :--- |
| L1: 静态快速筛查 | 所有附件 | 检查扩展名、双重扩展名、MIME 类型 | 速度极快 (<1s)，成本低 |
| L2: 动态沙箱分析 | 仅高风险附件 | 上传至 VirusTotal 分析 | 发现 0-day、宏病毒、无文件攻击 |
| L3: 综合决策 | 结合 L1 和 L2 结果 | 综合判定风险等级 | 平衡速度与安全性 |

#### 性能优化
- **文件大小限制**：>10MB 的文件跳过沙箱分析
- **低风险文件跳过**：纯图片、纯文本文件跳过沙箱分析
- **API 配额管理**：智能控制 API 调用频率，避免超出免费额度

#### 前端展示
- **沙箱分析状态**：在检测结果中显示沙箱分析状态
- **检测详情**：显示 VirusTotal 检测率和结果
- **风险等级**：基于静态和动态分析结果综合评定

## 安全性提升

### 1. 检测能力增强
- **零日威胁检测**：通过沙箱动态分析，能够检测未知的恶意文件
- **宏病毒检测**：能够发现隐藏在 Office 文档中的宏病毒
- **文件伪装检测**：即使文件扩展名被伪装，也能通过动态行为检测

### 2. 可靠性提升
- **API 容错处理**：沙箱 API 失败时自动降级为静态分析
- **异步处理**：支持沙箱分析的异步执行，避免阻塞用户界面
- **隐私保护**：在返回结果中移除附件内容，保护敏感信息

## 使用指南

### 配置 VirusTotal API
1. 访问 [VirusTotal](https://www.virustotal.com) 注册账号
2. 获取 API 密钥
3. 在系统设置中配置 API 密钥
4. 保存配置后即可启用沙箱分析功能

### 分析流程
1. 上传或粘贴邮件内容
2. 系统自动执行静态分析
3. 对可疑附件进行沙箱分析
4. 生成综合检测报告
5. 在报告中查看沙箱分析结果

## 技术参数

- **静态分析时间**：< 1 秒
- **沙箱分析时间**：30 秒 - 2 分钟（取决于文件大小和 VirusTotal 响应速度）
- **API 限制**：VirusTotal 免费 API 每分钟 4 次，每天 500 次
- **支持文件类型**：可执行文件、脚本、Office 文档、PDF、压缩包等

## 兼容性
- 与现有系统完全兼容
- 无需修改现有代码
- 可在无 API 密钥情况下正常运行（仅使用静态分析）
